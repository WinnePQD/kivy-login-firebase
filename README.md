# 📱 Aplicativo de Login com Firebase (KivyMD)

## 📌 Sobre o projeto

Este é um aplicativo mobile desenvolvido em Python utilizando KivyMD, com foco em autenticação de usuários.

O app permite que usuários realizem login utilizando e-mail e senha, com validação dos dados através do Firebase Realtime Database.

## 🚀 Funcionalidades

* Login com e-mail e senha
* Validação de credenciais via Firebase
* Feedback ao usuário (Toast no Android e Popup no desktop)
* Redirecionamento após login
* Tela de cadastro de usuário
* Tela de recuperação de senha
* Estrutura preparada para login com Google (em desenvolvimento)

## 🛠 Tecnologias utilizadas

* Python
* Kivy / KivyMD
* Firebase Realtime Database
* Requests (requisições HTTP)
* Pyjnius (integração com Android)

## 📱 Plataformas

* Android
* Desktop (modo de teste)

## ⚠️ Observações

Este projeto foi desenvolvido como estudo/prática de autenticação de usuários em aplicativos mobile utilizando Python.
